### 简介
项目名称：闲置Kindle再就业 [开发中]
简介：一个带图片封面的日历，一个极简的时钟，一个开盲盒番茄钟；
可以在页面中打卡上下班，记录每日工作时长（没班硬上，没苦硬吃）；
一个 Vibe Coding 项目

### 主要页面和功能

1. 日历页
- 封面图：每次刷新页面/每日凌晨自动刷新，随机显示一张封面图片。封面图来自pixabay.com 经过AI处理后形成墨水屏风格。
- 日历：其实是月历。封面排版参考小🍠上的皆川明日历排版（实际替换了pixabay上的图片）皆川明官网 www.mina-perhonen.jp
- 今日工位：每日首次点击【上班打卡】按钮，工位+1；工位人数包含上班打开的自己，和通过完成番茄钟获得的盲盒角色
- 假装打卡：首次点击【上班打卡】开始记录时间；点击【下班打卡】暂停计时；再次点击【上班打卡】继续计时。每日凌晨自动清零；当前版本关闭浏览器后会清零。

2. 时间页
- 时间：一个极简的时钟
- 根据IP推测定位，并获取天气
- 一句"搬砖与摸鱼的赛博哲学"
- 三只随地大小吃草的羊

3. 番茄钟
- 番茄钟：中杯、大杯、超大杯，对应三种专注时长：25+5min  50+10min  90+15min
- 完成一次番茄钟，解锁一个盲盒；下方工位+1，增加一个"盲盒"角色
- "盲盒"展示：所有的角色在工位上站一排（虽然显示在很不起眼的地方，让我联想到自己电脑桌面上摆放的一排皮卡丘）
